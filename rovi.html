Roxio Easy VHS to DVD 3

Roxio Easy VHS to DVD is software that provides easily transfers videos from VHS, Hi8, V8 or analog camcorders to DVD via the included USB capture device.

Achievements

  • Verified the compatibility with different media under different input formats and output formats (video, audio and images) for transformation test.
  • Improved the accuracy of the language software and online help, messages, interface resources, command key sequence as consistency checks.


Roxio Game Capture

Roxio Game Capture is a software that provides the ability to capture real-time Gameplay from Xbox 360 or PS3 to PC when continue playing in HD on television and also enables user to edit and share the captured gameplay.

Achievements

  • Helped deliver 3 products: Roxio Game Capture SP1, SP2 and SP3.
  • Tested the software recording capabilities with different recording devices, recording media of different games.
  • Improved the accuracy of product online help, pop-up messages and hotkey sequences.
